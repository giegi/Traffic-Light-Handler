TLH CONSTRUCTOR


window.kw_tlh.tlhObject = new tlhl(objectLabel, configObject);

objectLabel - { String }

configObject - { Object }
{
// TLH Callback - {default: null | Function | String }
'cb': null,
// TLH Library Url - {default: undefined | String }
'libUrl': undefined,
// TLH Library Success Callback - {default: null | Function }
'libCbSuccess' : null
// TLH Library Error Callback - {default: null | Function }
'libCbError' : null
// TLH Continue On Failure - {default: "true" | String(true|false) }
'cof' : "true"
}



METHODS



getName() Returns TLH OBJECT'S NAME

return String



getBlockers(getMode) Prints into the console all the dependencies and their state for this TLH object

getMode { null | "tlh" | "library" }

return null



getStatus(doCallback) Prints into the console TLH object status. If doCallback is true, it performs a test to the object to verify if all the dependencies are satisfied. If yes it will fire the object callback

doCallback { false | Boolean }

return Boolean



addRedLight(_light, _status) Adds a dependency to the TLH object

_light { not empty | String }
_status { false | tlhObject | Boolean | Function (not implemented yet) }

return true if success



addLibRedLight(_light, _status) Adds a dependency blocking the library downloading to the TLH object

_light { not empty | String }
_status { false | tlhObject | Boolean | Function (not imp
return true if success



setLibGreenLight(_light, _checkStatus, _doCallback) Declares satisfied a dependency for library downloading. If _checkStatus is true, a check will be performed on the TLH object. If _doCallback is true, and the dependencies are satisfied, the object callback will be fired

_light { not empty | String }
_checkStatus { not empty | Boolean }
_doCallback { false | Boolean }

return String



setGreenLight(_light, _checkStatus, _doCallback) Declares satisfied a dependency for TLH object. If _checkStatus is true, a check will be performed on the TLH object. If _doCallback is true, and the dependencies are satisfied, the object callback will be fired

_light { not empty | String }
_checkStatus { not empty | Boolean }
_doCallback { false | Boolean }

return String



removeLight(_light) Removes an object dependency from TLH object

_light { not empty | String }

return true if success



removeLibLight(_light) Removes a library dependency from the TLH object

_light { not empty | String }

return true if success
